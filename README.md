## Overview
This repository showcases the **Sales Module** developed for the BIGS Core ERP system, built on the **PHP Yii 2 Framework**. This project was completed during my internship and highlights my expertise in full-stack web application development, database integration, and UI design within a complex, multi-module enterprise environment.

## üéØ My Core Contributions: Sales Module

The Sales Module is a critical tool for managing the entire sales lifecycle, from lead generation to contract closure. My responsibilities included:

| Feature | Description | Key Technologies Used |
| :--- | :--- | :--- |
| **Sales Dashboard** | Provides a real-time summary of sales performance, key metrics, and weekly/monthly targets. | Yii 2, Chart.js, Custom SQL views |
| **Client Management (CRUD)** | Full Create, Read, Update, Delete (CRUD) functionality for managing potential client contacts and leads. | PHP, MySQL (ActiveRecord), Yii 2 Forms |
| **Sales Pipeline (Kanban)** | Visual Kanban board implementation to track leads through different stages (e.g., Prospecting, Qualification, Proposal, Closed-Won). | JavaScript (drag-and-drop), AJAX |
| **Quotation & Contract Management** | Functionality to generate, send, and track quotations and formal contracts for closed deals. | PDF Generation Library, Yii 2 Controllers |
| **Sales Reporting** | Generates detailed reports on sales activity, pipeline value, and conversion rates. | Custom report views, Data filtering |
